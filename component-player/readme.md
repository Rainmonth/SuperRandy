组件说明

媒体播放组件，主要提供音视频的播放，同时提供一些简单的界面实现；

### 视频播放

采用Ijk、Exo和系统三种播放内核来进行视频播放，三种内核可灵活配置，采用Ijk时注意获取相应的`.so`文件，不然会报错

- 翻页播放
- 详情播放
- 播放特效
- 带广告播放
